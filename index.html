    📖 About This Project

    Advance Study Planner

    A smart, browser-based scheduling tool designed to help students plan and track study sessions effectively.

    🧠

    Smart Allocation

    Time is distributed by difficulty weight — harder subjects get more hours automatically.

    ⏱️

    Session Timers

    Built-in countdown timers for each slot keep you focused and on schedule.

    📊

    Live Progress

    Real-time progress bars track completion per day and across all sessions.

    ☀️

    Time-of-Day Split

    Plans rotate between morning, afternoon, and night to prevent monotony.

    💾

    Auto Save

    Subjects persist via localStorage — no data lost on page refresh.

    🌗

    Dark & Light Mode

    Toggle between an atmospheric dark theme and a crisp light theme.

    How It Works

    Each subject is weighted by difficulty (Easy = 1, Moderate = 2, Hard = 3). Daily hours are divided proportionally so tougher subjects get more time. Sessions are then split across morning, afternoon, and evening in rotating patterns for variety.

    Technologies Used

    HTML5CSS3Vanilla JS Canvas APIlocalStorageCSS Grid Google FontsResponsive Design

    Purpose

    Built as an academic tool to help engineering students at Chitkara University manage exam prep and semester coursework more effectively — making planning simple, visual, and motivating.
